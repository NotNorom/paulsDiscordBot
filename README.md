# paulsDiscordBot

-> 	Before being able to start the bot,
	create a config.json file like this:

	{
	"token": "YOUR_TOKEN_HERE",
	"prefix": "YOUR_PREFERED_PREFIX"
	}


-> Navigate to the project in your console
$cd {your path}/paulsDiscordBot

-> Initialize the bot
$npm install

-> Start the bot
$node .
